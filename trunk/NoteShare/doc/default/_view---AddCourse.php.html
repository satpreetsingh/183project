AddCourse.php

Responsible for displaying the add course view. Most of the visual content is driven by a combination of php functions. The combo boxes where the user can select the universities, depts, courses, etc. are driven by AJAX calls. The XML response is converted to options within the combo boxes via XSLT.

The expect way to generate an add course form is this sequence: openEditor() -- once, first genCmbBox() -- several genButtons() -- once, after the combo boxes closeEditor() -- once, last

See ajaxView.php for the AJAX implementation.
